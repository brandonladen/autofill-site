Mobile Installation (Android)

First things first - you'll need Quetta Browser to run extensions on Android!

Download Quetta Browser from Play Store

Installation Steps

  1. Download the ZIP file using the button above. The file saves to your Downloads folder.
  2. Open Quetta Browser and tap the three-dot menu (bottom right).
  3. Go to Extensions - tap the Extensions icon (puzzle icon) from the menu.
  4. Manage Extensions - tap "Manage Extensions" at the bottom.
  5. Developer options - tap "Developer options" button.
  6. Load extension - tap "(from .zip/.crx/.user.js)"
  7. Select the ZIP file: Search for "extension.zip" and select the one downloaded recently (check the date).
  8. Done! Open any Moodle quiz page - the yellow lightbulb widget appears automatically.

Finding the ZIP file on phone

Open your file manager and check: Internal Storage > Download. Or search for "extension.zip" in the file picker.

Widget not showing?

Make sure the extension is enabled in Extensions page, then reload the quiz page.

Desktop Installation (Chrome/Edge/Brave)

Install the extension manually on your computer:

  1. Download the ZIP file using the button above.
  2. Extract the ZIP file to a folder (right-click > "Extract All" on Windows, or double-click on Mac).
  3. Open Chrome and type this in the address bar:
    chrome://extensions
  4. Enable Developer Mode - toggle the switch in the top-right corner.
  5. Click "Load unpacked" button (top-left area).
  6. Select the extracted folder - choose the folder containing manifest.json.
  7. Done! The extension icon appears in your toolbar.

Keep the folder!

Don't delete the extracted folder - Chrome needs it to run the extension. Move it somewhere permanent like Documents.
