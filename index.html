Mobile

Install on Phone

You need a Chromium-based mobile browser that supports extensions. Pick your device below.

Android — Quetta Browser

  1. Download Quetta Browser from the Play Store.
    Download Quetta
  2. Tap below to open the LaceFi Autofill page in Quetta.
    Install Extension
    When asked which browser to use, tap Quetta and select Always. Android will remember your choice.
  3. Tap Add to Chrome.
  4. Log in your portal and open any quiz page. The yellow lightbulb appears.

iOS — Orion Browser

  1. Download Orion Browser by Kagi from the App Store.
    Download Orion
  2. Tap below to open LaceFi Autofill in Orion.
    Open in Orion
    If iOS asks "Open in Orion?" tap Open.
  3. Tap Add to Orion.
  4. Log in your portal and open any quiz page. The yellow lightbulb appears.

Widget not showing?

Check your browser's Extensions page to make sure LaceFi Autofill is enabled, then reload the quiz page.

Desktop (Chrome / Edge / Brave)

Install on PC

  1. Visit the Chrome Web Store page.
  2. Click Add to Chrome.
  3. Open any quiz page. The lightbulb widget appears.
  4. Optional: Pin the extension to your toolbar (click puzzle icon, then pin).
