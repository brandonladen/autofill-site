02

Install on phone

Android

Quetta
  1. Download Quetta Browser from the Play Store. Download Quetta
  2. Open LaceFi Autofill in Quetta. Install Extension When asked which browser to use, tap Quetta and select Always. Button didn't work? Copy the link and paste it in Quetta.
  3. Tap Add to Chrome.
  4. Log in to your portal and open any quiz — the lightbulb appears.

iOS

Orion
  1. Download Orion Browser by Kagi from the App Store. Download Orion
  2. Open LaceFi Autofill in Orion. Install Extension If iOS asks “Open in Orion?”, tap Open. Button didn't work? Copy the link and paste it in Orion.
  3. Tap Add to Orion.
  4. Log in to your portal and open any quiz — the lightbulb appears.

Desktop

Chrome · Edge · Brave
  1. Visit the Chrome Web Store page. Add to Chrome
  2. Click Add to Chrome, then open any quiz page — the lightbulb widget appears.
  3. Optional: pin the extension to your toolbar (puzzle icon → pin).
Widget not showing?

Open your browser's Extensions page, make sure LaceFi Autofill is enabled, then reload the quiz page.
