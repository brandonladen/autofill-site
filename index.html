LaceFi Autofill

Autofill practice quizzes and forms on Moodle, desktop & mobile (Quetta)

Mobile (Quetta Browser)

Follow these simple steps to install the extension from the Chrome Web Store on your Android device using Quetta Browser:

  1. On your Android device, download Quetta Browser from the Google Play Store and open the browser if it is not already installed.
  2. In Quetta Browser, visit the LaceFi Autofill Chrome Web Store page.
  3. Tap Add to Chrome to install the extension directly from the Chrome Web Store.
  4. Open a quiz in this browser and you are good to go! The floating Autofill widget, a lightbulb icon, will appear on quiz pages.

If the widget does not appear

  1. Open Quetta's Extensions list and verify LaceFi Autofill is enabled.
  2. Reload the Moodle quiz page in Quetta (make sure you have opened a cat/exam).

Desktop (Chrome)

Follow these simple steps to install the extension from the Chrome Web Store:

  1. Visit the LaceFi Autofill Chrome Web Store page.
  2. Click Add to Chrome to install the extension.
  3. After the extension installs, open a quiz. The floating Autofill widget, a yellow lightbulb icon, appears only when you open a quiz page.
  4. To make the extension easy to find, pin its icon to the Chrome toolbar: click the puzzle icon in the toolbar, then click the pin next to the LaceFi Autofill entry.

How to use the widget

  1. Tap the yellow floating lightbulb to open the Autofill panel or open the extension popup from the toolbar.
  2. Enter your license key in the Enter License field or purchase credits in the popup as instructed.
  3. Use Autofill only for practice and study. See the Compliance section below.

Licenses and payments

The system uses Paystack for secure payments and issues license keys. You can also receive manual license keys from your instructor or admin.

  • After purchase, paste the license into the extension popup or the floating widget's "Enter License" modal.
  • The extension rechecks the license with the backend before using credits.

Support & privacy

We only store license keys and minimal usage metadata on the server to track credits. Payment card data is handled by Paystack.
