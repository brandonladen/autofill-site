Mobile Installation

Install the extension on your Android device using Quetta Browser:

  1. Download and install Quetta Browser from the Google Play Store if you don't have it already.
    Download Quetta
  2. Open Quetta Browser and visit the LaceFi Autofill Chrome Web Store page.
  3. Tap Add to Chrome to install the extension.
  4. Open any quiz page — the floating lightbulb widget will appear automatically.

Widget not showing?

Open Quetta's Extensions list to verify LaceFi Autofill is enabled, then reload the quiz page.

Desktop Installation

Install the extension directly from the Chrome Web Store:

  1. Visit the LaceFi Autofill Chrome Web Store page.
  2. Click Add to Chrome to install.
  3. Open any quiz page — the yellow lightbulb widget appears automatically.
  4. Optional: Pin the extension to your toolbar by clicking the puzzle icon, then clicking the pin next to LaceFi Autofill.
    Extension icon

How to Use

  1. Tap the floating lightbulb or click the extension icon in your toolbar.
  2. Enter your license key in the popup or purchase credits as instructed.
  3. Use Autofill only for practice and study purposes.

Licenses & Payments

The system uses Paystack for secure payments and automatic license key generation.

  • After purchase, your license key is automatically saved to the extension.
  • The extension validates your license with the system before using credits.
  • You can also receive manual license keys from your instructor or admin.

Privacy & Support

We only store license keys and minimal usage metadata to track credits. Payment card data is handled securely by Paystack — we never see your card details.
